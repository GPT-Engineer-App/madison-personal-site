# madison-personal-site

Create a personal website for James Madison with the following features:
- A professional-looking homepage with a brief biography.
- A gallery section to showcase images.
- A contact form for inquiries.
- Navigation bar with links to different sections of the site.
Use placeholders for the biography and images.
